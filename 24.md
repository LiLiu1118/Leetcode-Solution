#  24. Swap Nodes in Pairs

```c++
class Solution {
public:
    ListNode* swapPairs(ListNode* head) {
        
        if (!head || !head->next){
            return head;
        }
        ListNode* tmp = head->next;
        head->next = swapPairs(tmp->next);
        tmp->next = head;
        return tmp;        
    }
};
```

# 思路

递归算法，递归算法复杂度为递归的深度乘以每次递归的操作次数。
