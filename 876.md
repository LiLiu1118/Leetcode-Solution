#  876. Middle of the Linked List

```c++
class Solution {
public:
    ListNode* middleNode(ListNode* head) {
        ListNode* slow = head;
        ListNode* fast = head;
        for (; fast && fast->next; fast = fast->next->next){
            slow = slow->next;
        }
        return slow;
    }
};
```
