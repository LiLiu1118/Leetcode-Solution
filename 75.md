#  75. Sort Colors

```c++
class Solution {
public:
    
    void sortColors(vector<int>& nums) {
        int low = 0, mid = 0, high = nums.size() - 1;
        //使用int而不是size_t 因为size_t是unsigned的，high--有可能会为-1
        while (mid <= high){
            switch (nums[mid]){
                case 0: swap(nums[low++], nums[mid++]); break;
                    
                case 1: mid++; break;
                    
                case 2: swap(nums[mid], nums[high--]); break;                   
            }
        }
    }
};
```

# 思路

保证左边为0，右边为2,中间三者都可能。
